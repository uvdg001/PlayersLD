# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1leK7ApJP6YZcX7VeD1v3YqUe_d7Pg0KW

## Run Locally

**Prerequisites:**  Node.js

1.  **Install dependencies:**
    `npm install`

2.  **Set up Environment Variables:**
    *   Create a file named `.env` in the root of your project.
    *   Copy the contents of `.env.example` into your new `.env` file.
    *   Fill in the values for your Firebase project configuration. You can find your Firebase config in your Firebase project settings.

3.  **Run the app:**
    `npm run dev`
