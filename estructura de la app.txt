Para la próxima vez que quieras cambiar algo (un color, un texto, agregar un jugador), el proceso es súper rápido:
Haces el cambio en tu código y guardas.
Abres la terminal y escribes:
code
Bash
npm run build
(Esto actualiza el "paquete" con los cambios nuevos)
Luego escribes:
code
Bash
firebase deploy
(Esto sube la actualización a internet)
¡Y listo! En segundos tus amigos verán el cambio.
